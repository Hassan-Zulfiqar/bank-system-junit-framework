# Bank System - JUnit 5 Automation Framework

## Features
- JUnit 5 Testing
- Maven Build Tool
- CSV Data-Driven Testing
- Excel Data-Driven Testing (Apache POI)
- Test Suite Execution
- Parallel Testing

- com.bank.service

2. Run TestSuite.java
